Introduction

The purpose of this project is to create a repository for all known GRB/SNe associations. There has been limited research in this area. This tool is inteneded to improve access to new associations by aggregating the data in one place. Bulk parameters of the GRB and Supernova will be reported here, along with the lightcurve data.
