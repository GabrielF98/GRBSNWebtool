Help with Graphs on Event Pages

{% endblock %} {% block content %}

These graphs have several features to improve useability, these can be found in the toolbar at the top right of each plot. Tools include:

  • Pan.

  • Zoom in on box.

  • Scroll wheel zoom.

  • Save plot.

  • A reset button to reset the changes made with any of the tools.

  • Hover over a point with the mouse to see information about that point.

  • On the Spectrum plot you can click on the legend to mute a particular line in order to see other data more clearly.
