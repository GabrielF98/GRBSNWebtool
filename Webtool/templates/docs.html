Introduction

This tool was developed to allow astronomers to more easily share and evaluate data for GRB-SN associations, by bringing many sources and types of data together in one place.

GRBSN webtool was created by Gabriel Finneran, Laura Cotter and Antonio Martin-Carrillo at University College Dublin. An in depth look at the tool can be found in this paper.

Sources of Data in this Webtool

NASA ADS: This webtool used NASA’s Astrophysics Data System to gather observational data and calculated quantities for each event in the catalogue. These papers provide much of the data in the webtool. They are referenced at the bottom of their respective pages and these references are also provided when downloading any of the data.


The Open Supernova Catalogue: The Open Supernova Catalogue's API feature was used to gather the bulk of the graphical data for this tool, in particular the optical photometry and spectra. A detailed description of this tool can be found in Guillochon et al. 2017


Swift Telescope UK Site: Provided the XRT Xray plot data as well as the BAT RA and Dec positions.


The Transient Name Server (TNS): was used to create the plot on the homepage showing the number of events per year.

Contributing Data to the Webtool

Details on how to upload your own data or to flag issues with missing or bad data can be found on our Github repository.

Citing this work

If your work has made use of the GRBSN webtool, please cite this paper. We would also appreciate if you could add the following acknowledgement (or similar) to your work: such as:

''This research has made use of the GRBSN webtool, available at https://grbsn.watchertelescope.ie''
